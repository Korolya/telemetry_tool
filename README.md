# Telemetry Tool

Консольная программа на Rust для сжатия, восстановления и анализа CSV-телеметрии.

## Сборка
Убедитесь, что установлен Rust (https://rustup.rs/). Затем в корне проекта выполните:

```bash
cargo build --release
```

## Запуск

### Анализ

```bash
cargo run --release -- analyze Telemetry_01_10_2023.csv report.txt
```

### Сжатие

```bash
cargo run --release -- compress Telemetry_01_10_2023.csv archive.bin
```

### Восстановление

```bash
cargo run --release -- decompress archive.bin restored.csv
```
